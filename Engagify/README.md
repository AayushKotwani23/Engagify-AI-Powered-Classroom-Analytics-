# Engagify – AI-Powered Classroom Analytics

## Overview
Engagify is an AI-powered classroom analytics system designed to automate attendance management and monitor student engagement in real time using computer vision and deep learning.

## Features
- Face Recognition Based Attendance
- Real-Time Emotion Detection
- Student Attentiveness Tracking
- Attendance Report Generation
- Student Database Management

## Technologies Used
- Python
- OpenCV
- DeepFace
- TensorFlow
- Pandas
- NumPy
- Matplotlib

## How It Works
1. Captures live video from webcam.
2. Recognizes students using face recognition.
3. Detects emotions and engagement levels.
4. Marks attendance automatically.
5. Generates attendance records.
